# MathApp (beta version)

A **2D math game** built with Unity and C#. Practice basic arithmetic operations in a fun and interactive way.

##  About

MathApp is a 2D Unity game designed to make learning basic math engaging. Players face arithmetic challenges involving addition, subtraction, multiplication and division, with instant visual feedback.

##  Features

- **4 operations** — addition (+), subtraction (−), multiplication (×) and division (÷)
- **2D gameplay** — interactive Unity environment
- **Instant feedback** — visual response to correct and incorrect answers
- **Progressive difficulty** — challenges adapt as the player advances

## How to Run

### Option 1 — Play directly
Download the latest build from [Releases](https://github.com/nik444ds/MathApp/releases) and run the executable.

### Option 2 — Open in Unity
1. Clone the repository:
   ```bash
   git clone https://github.com/nik444ds/MathApp.git
   ```
2. Open the project in **Unity 2021.3+** (or the version used)
3. Open the main scene and press **Play**
